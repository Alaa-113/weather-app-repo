WeatherApp

WeatherApp is a modern web platform that lets users check the current weather and forecast for cities around the world. It provides real-time weather data in a clean and interactive interface, helping users plan their day or trips more efficiently.

Project Idea

Weather is essential for daily planning, travel, and outdoor activities, but many weather apps are either cluttered, slow, or lack detailed forecasts.

WeatherApp solves this by combining a user-friendly Angular frontend with a Django backend that fetches live data from a weather API, delivering accurate and instant weather information.

Tech Stack
Layer	Technology
Frontend	Angular
Backend	Django
Database	PostgreSQL
Features
Search weather by city
Display current weather: temperature, humidity, wind speed, and conditions
Optional: 3–5 day weather forecast
Responsive design for desktop and mobile
