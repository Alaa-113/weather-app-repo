# 🌤 WeatherApp

**WeatherApp** is a web platform that shows the current weather and a simple forecast for any city. It combines a clean Angular frontend with a Django backend that fetches live weather data.

---

## Tech Stack

| Layer      | Technology |
|------------|------------|
| Frontend   | Angular    |
| Backend    | Django     |
| Database   | PostgreSQL |

---

## Features

- Search weather by city  
- Show current temperature, humidity, wind, and conditions  
- Optional: 3–5 day forecast  
- Responsive design
